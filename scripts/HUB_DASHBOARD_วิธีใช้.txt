แท็บนี้คืออะไร?
--------------
หน้าดูอย่างเดียวที่รวมรหัส PTP จาก「ชีตสำหรับทำงาน」+ RXT จาก「ทรัพย์ Hub」
เรียงใหม่→เก่า · คอลัมน์ 16 ช่อง:

รหัส | ที่มา | วันที่ | โครงการ | ประเภท | ห้อง | ตรม. | ชั้น | เช่า | ขาย | ทำเล | สถานี | ต้นทาง | เจ้าของ | ที่โพสต์ | เพจ

ประเภท = จากคอลัมน์「ประเภท」ชีทหลัก (Condo→คอนโด · House→บ้าน · ที่ดิน/land→ที่ดิน · อื่นๆ แสดงตามต้นทาง)
ลิงก์ = จากชีทหลัก/Hub:
- ต้นทาง ← ลิ้งค์ต้นโพสต์
- เจ้าของ ← เฟสเจ้าของ
- ที่โพสต์ ← ลิ้งค์โพส
- เพจ ← ลิ้งค์โพส Pages

คอลัมน์ลิงก์แสดงเป็น「เปิด」(คลิกได้) · ตัวเลข ตรม. / เช่า / ขาย รูปแบบเดิม
แช่แข็งแถว 1–5

แถวบนสุดของแท็บ
---------------
- A1: ชื่อหน้า Property Hub · ทรัพย์รวม …
- แถว 2: 「ค้นหาทั่วไป」+ ช่องเหลือง C2 (รหัส / ชื่อโครงการ เท่านั้น)
- แถว 3: 「ค้นหาทำเล/BTS」+ ช่องเหลือง C3 (ทำเล + สถานี เท่านั้น)
- A4: สถานะ / จำนวนรายการ
- แถว 5: หัวคอลัมน์ (ตัวอักษรขาวบนพื้นเขียว)
- A6 ลงไป: ข้อมูล (สูตรสด หรือสคริปต์รีเฟรช)

ใช้งาน
------
- เปิดแท็บแล้วเห็นรายการทั้งหมดทันที (เรียงใหม่→เก่า) โดยไม่ต้องพิมพ์ค้นหา
- ว่างทั้ง C2+C3 = แสดงทั้งหมด
- กรอก C2 อย่างเดียว = กรองรหัส/โครงการ (เช่น 「ทรู」→ Thru Thonglor เท่านั้น — ไม่ดึง True Digital Park จากทำเล)
- กรอก C3 อย่างเดียว = กรองทำเล/BTS
- กรอกทั้งคู่ = AND (ต้องตรงทั้งสองเงื่อนไข)
- คลิก「เปิด」ในคอลัมน์ลิงก์เพื่อเปิดต้นทาง / เจ้าของ / โพสต์ / เพจ
- วันที่ไม่ใช่วันที่จริง (เช่น 「เว้น」) จะถูกดันไปท้าย ไม่ลอยบนสุด
- หรือเมนู Property Hub → รีเฟรชหน้าทรัพย์รวม (ถ้าติดตั้ง Apps Script แล้ว)
- แถวสีเขียวอ่อนที่「ที่มา」= จากชีท (PTP) · สีฟ้าอ่อน = จาก Hub (RXT)

ติดตั้ง Apps Script (ครั้งเดียว — ทางเลือก)
------------------------------------------
1. เปิดชีททดลองด้านบน
2. เมนู Extensions → Apps Script
3. วางไฟล์ scripts/google_apps_script_hub_dashboard.gs → Save
4. รัน setupAppDashboard → Allow
5. ได้แท็บสไตล์เขียว · ช่องค้นหาเหลือง C2+C3 · หัวตาราง 16 คอลัมน์ · ล็อกแก้ได้แค่ C2/C3

ซิงค์ทำเล / BTS / ลิงก์ จากแอป
-------------------------------
- ในแอปกด「ซิงค์ไปชีท Hub」→ เขียนแท็บ「ทรัพย์ Hub」ใหม่ทั้งแผ่น
- Hub sync ส่ง ประเภท · ลิ้งค์ต้นโพสต์ · เฟสเจ้าของ · ลิ้งค์โพส · ลิ้งค์โพส Pages ตาม prop_to_hub_row
- ทุกครั้งจะดึงทำเล + สถานีรถไฟฟ้า จาก master โครงการในแอป
- แท็บ `_proj_loc` (ถ้ามี) = lookup ชื่อโครงการ→ทำเล/สถานี สำหรับแถวชีทหลัก
- อัปเดตสูตรสด: python3 scripts/_live_fix_app_dash_v8_links.py

หมายเหตุ
--------
- แท็บ「ทรัพย์ Hub」ยังเป็นที่ซิงค์จากแอป (เขียนได้)
- แท็บ「ทรัพย์รวม · แอป」= หน้าดูอย่างเดียว คล้ายหน้าหลักในแอป (มีประเภท + ลิงก์แล้ว)
- สูตรอ้างอิง: data/sheet_sync_demo/dashboard_formula.txt
