แท็บ「ฟอโล่วเคส」— ใช้คู่กับเมนูฟอโล่วใน Property Hub
====================================================

ในแอป (http://127.0.0.1:8765/)
- เมนูซ้าย「ฟอโล่ว」→ กรอกเคสลูกค้า
- ปุ่ม「แนะนำทรัพย์」จับคู่จากลิสต์ใน Hub ตามงบ / ทำเล / BTS / ห้องนอน
- กด「+ เสนอแล้ว」หรือ「+ นัดดู」เพื่อติดรหัสห้องเข้าเคส
- 「ส่งออก CSV ชีท」→ ไฟล์ data/customer_followup_export.csv

วางลงชีททดลอง
1. เปิดชีททดลอง → แท็บใหม่ชื่อ「ฟอโล่วเคส」
2. File → Import → Upload customer_followup_export.csv
   หรือคัดลอกหัวตารางจากไฟล์นั้นวางแถวแรก
3. ใช้แอปเป็นหลัก · ชีทเป็นสำเนา/แชร์ทีม (ยังไม่ซิงค์อัตโนมัติกลับชีทจนกว่าจะมี Service Account)

หัวคอลัมน์หลัก
รหัสเคส, ช่องทาง, ชื่อในแชท, เบอร์, LINE ID, ผู้รับผิดชอบ, ประเภทดีล,
วันติดต่อครั้งแรก, วันคุยล่าสุด, ทักซ้ำในอีกกี่วัน, วันฟอโล่วถัดไป, สถานะ,
ทำเล, BTS/MRT, งบต่ำ–สูง, ห้องนอน, โจทย์สั้นๆ,
รหัสที่เสนอแล้ว, รหัสที่นัดดู, รหัสจอง/มัดจำ, โน้ตฟอโล่วล่าสุด
